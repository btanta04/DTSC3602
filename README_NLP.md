# NLP Analysis Branch

**Branch:** `nlp-analysis`   
**Focus:** Natural Language Processing for fraud detection

## What This Branch Does

This branch focuses on adding smart NLP capabilities to our fraud detection system:

- **Smart Fraud Detection** - Better than simple keyword matching
- **Article Summarization** - Auto-generate executive summaries  
- **Trend Analysis** - Find emerging fraud patterns
- **Topic Modeling** - Identify common fraud categories

## How to Use

### Run the NLP Analysis:
```bash
python nlp_analyzer.py
